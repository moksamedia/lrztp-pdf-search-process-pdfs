Note that one occasionally uses གི་ཡོད། (mostly for volitional verbs) for oneself with non-volitional
verbs if it is habitual. ང་ལོ་རྟག་པར་ན་གི་ཡོད།

I get sick every year.

གི་ཡོད། does not indicate volition here, but habit!

And one uses གི་ཡོད་རེད། for others even if you've seen it yourself - if the action is done habitually!

(S)he does kora every morning.
